# Cloud SQL Backup

## Details

- **Type:** `CloudSQLBackup`
- **Scope:** project

## Properties

- **`Instance`**: The Cloud SQL instance this backup was taken from, which may no longer exist
- **`Location`**: The location of the backup
- **`Name`**: The full resource name of the backup
- **`State`**: The current state of the backup
- **`Type`**: The type of backup, e.g. FINAL or AUTOMATED
